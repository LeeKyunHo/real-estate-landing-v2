# Real Estate Landing Page

부동산 상담 및 분양 안내를 위한 **랜딩 페이지 프로젝트**입니다.  
React + Vite 기반으로 제작되었으며 모바일 환경에서도 사용할 수 있도록 반응형 구조로 구성되어 있습니다.

현재 목적은 다음과 같습니다.

- 부동산 상담 유도 랜딩페이지 제작
- 실제 분양/매물 정보를 쉽게 교체할 수 있는 구조 설계
- React 기반 프론트엔드 개발 및 배포 경험 축적

---

# 주요 기능

- Hero 메인 배너
- 스크롤 유도 애니메이션
- 섹션 기반 랜딩 구조
- 상담 문의 폼
- 우측 하단 전화 상담 버튼
- 반응형 UI

# English Summary

This project is a **real estate landing page built with React and Vite**.

Purpose of the project:

- Build a marketing landing page for real estate consultation
- Structure the project so images and content can be easily updated
- Gain frontend development and deployment experience

Main features:

- Hero banner section
- Scroll indicator
- Section-based landing layout
- Contact form
- Floating call button
- Responsive UI
